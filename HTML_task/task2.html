Block and inline ELements

  • Block ELements:

    A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.

  • List of Element:

    • <div>
    • <h1>
    • <p>
    • <ul>
    • <li>

  • Inline ELements:

    Inline elements display in a line. They do not force the text after them to a new line. An anchor (or link) is an example of an inline element.
    You can put several links in a row, and they will display in a line.

  • List of Element:

    • <span>
    • <a>
    • <stong>
    • <em>
